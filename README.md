# Inventory Reservation Engine

A premium developer portfolio case study for a fintech-inspired inventory reservation system built for high-concurrency e-commerce checkout flows.

The showcase presents a distributed backend architecture that prevents overselling during payment processing using Redis-style locking, reservation queues, expiry timers, atomic stock updates, and asynchronous confirmation workflows.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Node.js / Express.js / NestJS concept layer
- Redis, PostgreSQL, MongoDB, and Docker architecture references

## Features

- Animated checkout architecture flow
- Realtime inventory reservation simulation
- Interactive backend workflow cards
- Performance metrics dashboard
- Terminal-style backend log animation
- Futuristic dark fintech UI
- Responsive portfolio-ready layout

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Production Build

```bash
npm run build
```
